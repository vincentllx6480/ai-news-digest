🌍 国际动态

OpenAI GPT-5.6 全面开放，ChatGPT Work 进军企业办公

7月9日，OpenAI 正式向公众开放 GPT-5.6 系列三档模型：旗舰 Sol（$30/百万输出token）、均衡 Terra（$15）、轻量 Luna（$6）。同步发布的 ChatGPT Work 超级应用可自主生成文档、表格、PPT 和网站，直指 Anthropic Claude Cowork 和微软 Copilot。此前该模型因美国政府安全审查延迟近两周，经商务部AI标准与创新中心批准后才全面放行。

Anthropic 企业份额反超 OpenAI，马斯克公开"认错"转向

Ramp AI Index 5月数据显示 Anthropic 企业 API 市场份额达 34.4%，首次超过 OpenAI 的 32.3%。Claude Fable 5 在 SWE-Bench Pro（80% vs 64.6%）和 FrontierMath（87.8% vs 83%）上领先 GPT-5.6 Sol。Elon Musk 公开表示"我显然错了"，称 Anthropic 是"明确的AI领导者"，SpaceXAI 每月向 Anthropic 支付 12.5 亿美元算力费用，并已收购 Cursor 编程工具。

钛媒体 Yahoo Tech

微软 MAI 自研模型加速"去 OpenAI 化"，成本降至 1/10

Build 2026 大会上微软发布 7 款自研 MAI 模型，已在 Excel 和 Outlook 中替换 OpenAI 模型，每周处理数十万次 AI 请求。微软声称相比 GPT-5.5 实现 10 倍成本效率。此前 OpenAI-微软独家协议已于 2025 年重谈为非排他性合作（至 2032 年），GPT-5.6 仅为 Microsoft 365 Copilot 的"首选"而不再"独占"模型。

Meta 7月9日发布 Muse Spark 1.1 多模态编程 AI（100万 token 上下文），首次以付费 API 向企业开放；同步推出 Muse Image 图像生成模型。SpaceXAI 发布 Grok 4.5（Opus 级模型），马斯克在称赞 Anthropic 的同时继续推进自研路线，并完成对 Cursor 的收购，进一步整合 AI 编程工具链。

新GPT-5.6 Sol Ultra 一小时证明 50 年数学难题

OpenAI 7月10日宣布，GPT-5.6 Sol Ultra 在不到1小时内成功证明了图论领域悬而未决50多年的"循环双覆盖猜想"（Cycle Double Cover Conjecture）。模型同时调用64个子智能体并行工作，配备对抗智能体负责寻找漏洞，原本预留8小时计算时间，最终仅用约1小时。英国数学家 Thomas Bloom 评价"非常漂亮的证明"，但尚未经过正式同行评审。按 Sol 定价估算成本仅 $275-485。
