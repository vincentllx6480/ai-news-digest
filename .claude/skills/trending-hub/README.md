# 全网热点追踪 / trending-hub

## 简介

帮你把抖音、微博、B 站、快手、知乎、头条、百度等 7 个平台的热搜聚合在一起，省得一个个平台去看。

**核心价值**

一键获取 7 大平台热搜数据，小时级更新。按平台分类独立展示，每个平台最多 50 条热搜，支持按关键词/平台筛选，告别逐平台查看的痛苦。

**适用对象**

- 🔍 竞品监控 — 查看特定平台热搜，了解竞争对手动态
- 🎯 选题调研 — 按关键词筛选相关热搜，快速匹配选题方向
- 📊 数据分析 — 获取全平台原始热搜数据，进行深度分析
- 📰 舆情追踪 — 监控特定事件在各平台的热搜表现 ---

## 核心功能

- **全平台热搜展示**：7 大平台热搜独立展示，每个平台最多 50 条
- **多平台组合查询**：支持查询单个和多个指定平台热搜
- **关键词精准筛选**：支持按关键词筛选相关热搜
- **多时间范围查询**：支持实时、今日、昨日、本周等多种时间范围
- **完整榜单查看**：每个平台可查看全部 50 条热搜
- **订阅推送服务**：定时推送最新热榜/昨日热榜

### 数据说明

- **覆盖平台**：7 大平台（抖音、微博、B 站、快手、知乎、头条、百度）
